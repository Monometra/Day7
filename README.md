# CLASS DAY 7 - CSS Grid Layout Mini Test

A simple demonstration of CSS Grid layout techniques using images and GIFs.

## Overview

This project showcases how to create responsive grid layouts using CSS Grid. It features two distinct grid sections displaying various images in an organized gallery-style layout.

## Structure

```
Day7/
├── index.html      # Main HTML file
├── css/
│   └── style.css   # Grid layout styles
├── media/          # Images and GIFs
└── README.md
```

## Features

- **Two Grid Layouts**: Demonstrates different grid configurations
- **Grid Spanning**: Uses `grid-row` and `grid-column` with `span` to create varied cell sizes
- **Responsive Images**: Images fill their grid cells using `object-fit`
- **Rounded Corners**: Global border-radius for a modern look

## CSS Grid Concepts Used

- `display: grid`
- `grid-template-columns` / `grid-template-rows`
- `grid-gap`
- `grid-row` / `grid-column` spanning

## How to Use

Simply open `index.html` in your browser to view the grid layout.
